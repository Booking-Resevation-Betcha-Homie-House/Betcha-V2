Employee Role Privilege System Test

Role-Based Access Control Features:

  • ✅ Fetch role privileges from API: GET /role/{roleID}
  • ✅ Dynamic sidebar filtering based on user privileges
  • ✅ Hide unauthorized menu items automatically
  • ✅ Hide content sections on dashboard
  • ✅ Support for multiple privileges per user
  • ✅ Special handling for TS (Transaction Specialist) role

API Integration:

API Endpoint: https://betcha-api.onrender.com/role/{roleID}

Method: GET

roleID Source: localStorage.getItem('roleID')

  • ❌ PSR Summary section - HIDDEN
